## 简介
**文图易 (WenTuYi)** 是一款专为隐私保护设计的安卓输入法。它通过物理对抗算法，将用户输入的文字实时转换为带有动态噪点和几何扭曲的图片，从而绕过即时通讯工具（如微信、QQ）的自动 OCR 文本扫描。

### 核心特性
- **抗 OCR 算法**：动态随机噪点 + 字符级位移 + 行旋转，物理隔绝 AI 扫描。
- **本地化渲染**：所有图片生成均在手机本地完成，无需服务器，零流量消耗。
- **隐私第一**：**完全不申请网络权限 (No Internet Permission)**，确保您的明文数据永远不会离开手机。
- **极简体验**：在微信对话框输入后，一键转图，长按粘贴即可发送。

## Introduction
**WenTuYi** is an Android Input Method Editor (IME) designed for extreme privacy. It converts your text into anti-OCR images in real-time using physical adversarial algorithms (dynamic noise, character-level displacement, and line rotation), effectively bypassing automated content surveillance in messaging apps.

### Key Features
- **Anti-OCR Engine**: Dynamic noise and text distortion to defeat AI-based surveillance.
- **Local Rendering**: 100% on-device processing. No cloud, no servers.
- **Zero-Trust Privacy**: **No Internet permission required**. Your data stays on your phone.
- **Seamless UX**: One-tap conversion inside any chat app.
